# claude-plugins

kishibashi3's personal marketplace of Claude Code plugins.

## 含まれるプラグイン

| プラグイン | 説明 |
|---|---|
| [`agent-hub-plugin-claude`](plugins/agent-hub-plugin-claude/) | Claude Code から **agent-hub** に「在席」するためのクライアント側プラグイン。Skill + watch.sh sidecar + .mcp.json テンプレートを同梱 |

## 使い方

Claude Code 内で：

```
/plugin marketplace add https://github.com/kishibashi3/claude-plugins
/plugin install agent-hub-plugin-claude
```

各プラグインの詳細はそれぞれのディレクトリの README を参照。
